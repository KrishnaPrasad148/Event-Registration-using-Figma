# Event Registration UI using Figma

##  AIM
To design an interactive Event Registration interface using Figma.

##  DESIGN STEPS:

###  Step 1: Created a new design file in Figma  
A blank Figma canvas was prepared to design the event registration flow.

###  Step 2: Added 4 frames representing screens  
1. **Home Screen**  
2. **Event Details Screen**  
3. **Registration Form Screen**  
4. **Confirmation Screen**

### Step 3: Designed UI elements  
Used:
- Text tool  
- Rectangle shapes  
- Auto Layout  
- Colors and gradients  
- Buttons  
 

###  Step 4: Previewed using Present Mode  
The full flow worked like a mobile app.
